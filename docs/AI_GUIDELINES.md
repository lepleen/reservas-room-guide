# AI Development Guidelines

Este projeto é desenvolvido incrementalmente.

Nunca redesenhe o projeto inteiro.

Sempre preserve:

- autenticação
- banco
- APIs
- formulários
- regras de negócio

Implemente apenas a tarefa solicitada.

Nunca faça commits.

Nunca faça push.

Nunca modifique arquivos fora do escopo.

Sempre pare ao finalizar a tarefa.
